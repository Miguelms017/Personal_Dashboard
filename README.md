# Personal Dashboard
By Miguel Angel Martinez Serrato
---

## What is
This project is about a personal dashboard where you can organize your navigator tasks on a single place.

This project counts with:

- a bar of shortcuts fully costomizable
- a weather widget
- a notepad
- a music widget *[still on development]*
- a random shortcut section *[still on development]*
---
## How to set up shortcuts?

1. Write the title of your shortcut and select ok
2. Paste the URL and select ok.
3. your shortcut is ready to use.
---
## How to set up a profile image?

1. Go to settings
2. select to upload a file below the section "user image profile"
3. Press set profile.
4. Done
